Consumo de APIs Públicas (IBGE e Banco Central)

Projeto desenvolvido na formação de Analista de Dados da EBAC.

Objetivo

Consumir APIs públicas do IBGE e do Banco Central utilizando Python, realizar tratamento e validação dos dados, salvar os resultados em diferentes formatos e comparar o uso de APIs com técnicas de web scraping.

Atividades desenvolvidas

* Consumo da API do IBGE
* Consumo da API do Banco Central
* Utilização da biblioteca `requests`
* Conversão dos dados para DataFrame com Pandas
* Tratamento de dados
* Validação de tipos, valores nulos e duplicados
* Exportação para CSV
* Exportação para Parquet
* Análise sobre web scraping, LGPD, direitos autorais e APIs

Arquivos

* `pratique_ibge_bcb.ipynb` — Notebook principal
* `bcb_tabela.csv` — Dados extraídos do Banco Central
* `dados_tratados.csv` — Dados tratados em CSV
* `dados_tratados.parquet` — Dados tratados em Parquet

Tecnologias utilizadas

* Python
* Pandas
* Requests
* Google Colab
* APIs públicas (IBGE e Banco Central)
* GitHub
